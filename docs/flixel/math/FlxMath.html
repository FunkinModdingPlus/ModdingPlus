staticinlinelerp(a:Float, b:Float, ratio:Float):Float

Returns the linear interpolation of two numbers if ratio is between 0 and 1, and the linear extrapolation otherwise.

Examples:

lerp(a, b, 0) = a
lerp(a, b, 1) = b
lerp(5, 15, 0.5) = 10
lerp(5, 15, -1) = -5
